# REZIIZI

Social app MVP — **React + Vite + TypeScript** + **Supabase**.

## Quick start

```bash
npm install
cp .env.example .env   # add VITE_SUPABASE_URL + VITE_SUPABASE_ANON_KEY
npm run dev
```
